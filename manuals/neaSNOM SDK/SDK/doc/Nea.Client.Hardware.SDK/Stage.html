Deactivate Method

Deactivates the stage and allows others stages to be activated.

Syntax

public void Deactivate ()

Remarks

This stage cannot be used any more after deactivation. In order to reuse the stage, you need to activate a new stage instance, e.g. by calling Microscope.ActivateSampleStage.
