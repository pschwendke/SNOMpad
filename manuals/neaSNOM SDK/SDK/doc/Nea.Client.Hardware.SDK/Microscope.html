AutoApproach Method

Approaches the tip to the sample automatically.

Syntax

public void AutoApproach (double relativeSetpoint)

Parameters

relativeSetpoint
(0.0 < relativeSetpoint < 1.0)

Remarks

This method will block until the automatic approach ends or a time-out occurs after one minute.

Example

This example shows how to auto-approach with 80% setpoint.
Example
            var client = new Connection("nea-server");
            var microscope = client.Connect();
            microscope.AutoApproach(0.80); // Auto-Approach with 80%
