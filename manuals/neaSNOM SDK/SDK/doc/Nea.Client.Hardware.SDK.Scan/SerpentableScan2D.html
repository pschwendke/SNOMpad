SkipRows Property

Gets or sets the number of rows to be skipped.

Syntax

public uint SkipRows { get; set; }

See Also

SerpentableScan2D.TakeRows
SerpentableScan2D.SetSerpentScan(uint, uint)

Value

The number of consecutive rows that will be skipped.

Remarks

  • A value of 0 indicates a normal scan (no serpent mode).
  • Images from serpent scans result in a lower resolution than BasicScan.ResolutionRows.

Example

This example shows how to start a serpent AFM scan with 100x20 pixels:
Example
            var scan = microscope.PrepareAfmScan();
            scan.SetResolution(100, 100);
            
            scan.TakeRows = 1;
            scan.SkipRows = 4;
            
            scan.Start();
