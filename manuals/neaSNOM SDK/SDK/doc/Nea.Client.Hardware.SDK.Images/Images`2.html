Parameters

acronym
Acronym that specifies the channel e.g. "M1A", "O0A" or "Z".

Returns

Image data for specified channel or null if the channel does not exist.

Remarks

  • This method can be called after the scan ended as well as when still in progress.
  • Data for the specified channel is loaded in the background, but not before you call this method.
